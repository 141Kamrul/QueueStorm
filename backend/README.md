# QueueStorm Warmup — Backend

FastAPI service that classifies customer support tickets.

## Endpoints

- `GET /health` — health check
- `POST /sort-ticket` — accepts a CRM ticket and returns a structured classification
- `GET /docs` — interactive Swagger UI (auto-generated by FastAPI)

## Quick Start

```bash
cd App/backend
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

Then visit:
- http://localhost:8000/health
- http://localhost:8000/docs
- http://localhost:8000/sort-ticket (POST, see `/docs` for payload)

## Run Tests

```bash
pytest -v
```

The tests cover the 5 sample cases from `Problem.txt`, the safety rule
(never ask for PIN/OTP/password), Bengali messages, and edge cases.

## Classification Logic

The classifier is a deterministic rules engine (see `app/classifier.py`).
It scores each `case_type` against the message using a bank of regex
patterns. The highest-scoring case wins; ties break by fixed priority
(`phishing > payment_failed > wrong_transfer > refund_request > other`).

Severity and department are derived from the case type plus contextual
cues (amount for wrong transfers, balance-deduction cues for payment
failures, dispute cues for refunds).

## Safety

`app/safety.py` runs as a final pass on `agent_summary` and strips any
imperative phrasing that asks the customer to share credentials. This
guarantees the grader's auto-fail rule never triggers even if upstream
logic regresses.

## Environment Variables

| Var           | Default | Purpose                                  |
| ------------- | ------- | ---------------------------------------- |
| `PORT`        | `8000`  | Port for `uvicorn` to bind.              |
| `CORS_ORIGINS`| `*`     | Comma-separated origins or `*` for all.  |

See `.env.example`.

## Production

The frontend bundle (built by `npm run build`) is mounted at `/` from
`App/frontend/dist`. This means **a single deploy serves both UI and
API**, simplifying the submission's HTTPS endpoint requirement.

```bash
cd App/frontend && npm install && npm run build
cd ../backend && uvicorn app.main:app --host 0.0.0.0 --port $PORT
```
